Generated using the prompt in `/prompts/ugc_prompt_template.md`. Grouped by angle so you can pick based on which persona/campaign you're running.

## Angle 2: Skeptic-turned-convert

## Angle 3: Busy person / no time
10. "I have literally zero time and I still found a way to make this work."

## Angle 4: Curiosity / local discovery
13. "Okay I was today years old when I found this gym in Bangalore and I'm kind of upset nobody told me sooner."

**Usage tip:** Pair Angle 1 hooks with the main script (problem = "bulky ≠ aesthetic"). Angles 2-4 work well as standalone 15-second hook-only Reels that drive traffic to the main long-form ad.
